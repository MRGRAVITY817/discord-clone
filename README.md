# A Discord Clone, build with Phoenix LiveView

This is a simple Discord clone, build with Phoenix LiveView. It is a work in progress, and is being built as a learning project.  

## Features

- [ ] User authentication
- [ ] User registration
- [ ] User profile
- [ ] Create a server
- [ ] Create a channel
- [ ] Join a server
- [ ] Join a channel
- [ ] Send messages
- [ ] Receive messages
- [ ] Real-time updates
- [ ] Notifications
- [ ] User roles
- [ ] User permissions
